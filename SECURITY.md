# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public issue.**

Instead, please email [pierluigi.lenoci@gmail.com](mailto:pierluigi.lenoci@gmail.com) with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 72 hours. We will work with you to understand and address the issue before any public disclosure.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
